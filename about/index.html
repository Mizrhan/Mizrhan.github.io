About

현실 데이터의 기준을 맞추고, 결과를 설명하는 분석가를 지향합니다.

숭실대학교 정보통계·보험수리학과에서 통계적 방법론, 추정검정론, 실험계획, 범주형자료분석, 다변량분석 등을 공부했습니다. 프로젝트에서는 데이터 불균형, 분포 변화, 결측, 이종 데이터 병합처럼 교과서 밖에서 자주 만나는 문제를 직접 다뤘습니다.

분석할 때 중요하게 보는 것

데이터 기준 확인

데이터를 바로 모델에 넣기보다, 수집 경로, 측정 주기, 결측 패턴, 이상치 기준을 먼저 확인합니다.

지표와 모델의 목적

전체 정확도보다 도메인에서 중요한 지표가 무엇인지 확인합니다. 시정 예측에서는 CSI를 주요 지표로 사용했습니다.

결과의 이유 설명

성능이 오르거나 떨어졌다는 사실에서 끝내지 않고, SHAP, PDP, Wasserstein Distance로 원인을 확인합니다.
