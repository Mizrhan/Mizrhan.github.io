  # iFood Customer Segmentation Analysis

  iFood 고객 데이터를 활용해 연령대별 채널 이용, 자녀 수/소득 수준에 따른 할인 민감도, 캠페인 응답 고객의 특성을 분석했습니다.
</div>

## Summary

| 항목 | 내용 |
|---|---|
| 목적 | 고객 세그먼트별 채널 선호와 프로모션 반응 차이 분석 |
| 데이터 | iFood marketing campaign data |

| 산출물 | 고객 세그먼트별 인사이트와 마케팅 전략 제안 |

## Analysis Questions

1. 연령대에 따라 구매 채널 이용 패턴이 다른가?
2. 자녀 수와 소득 수준에 따라 할인 프로모션 활용도가 다른가?
3. 캠페인 응답 고객과 비응답 고객은 구매 금액, 웹사이트 방문, 할인 구매 측면에서 어떤 차이를 보이는가?

## Key Findings

- 30대와 40대에서 웹 구매 비중이 높게 나타났습니다.
- 20대는 매장 구매 비중이 상대적으로 높게 나타났습니다.
- 50대 이상에서는 카탈로그 이용 비중이 상대적으로 높았습니다.
- 자녀 수가 많을수록 할인 프로모션 이용 비중이 높았습니다.
- 소득 수준이 높아질수록 할인 구매 비중은 낮아지는 패턴을 보였습니다.
- 캠페인 응답 고객은 총 구매 금액이 더 높게 분포했습니다.

## Recommendation

- 연령대별 채널 선호에 맞춰 프로모션 채널을 차별화합니다.
- 다자녀·저소득층 고객에는 할인 중심 전략을 정교화합니다.
- 캠페인 응답 고객과 비응답 고객을 같은 방식으로 대하지 말고, 구매 금액과 할인 민감도를 기준으로 세그먼트별 액션을 설계합니다.
