RAG and Public Service Idea

국방 모집병 정보 RAG 챗봇

예비용사들이 모집병 정보를 찾기 어렵다는 문제를 바탕으로, 병무청 모집병 정보를 문서 DB화하고 자연어 질문으로 지원 자격, 제출 서류, 커트라인 등을 확인할 수 있는 RAG 챗봇 서비스를 제안했습니다.

RAGLangChainLangGraph BM25FAISSFastAPI

역할

  • 모집병 정보 탐색의 어려움을 문제로 제시하고, RAG 챗봇이라는 해결책을 기획했습니다.
  • RAG 모델이 사용할 문서 DB 구축을 위해 모집병 정보 문서를 수집했습니다.
  • LangChain과 Hugging Face 모델 기반 문서 청킹, 벡터 임베딩 구상에 참여했습니다.
  • AI 시스템 개발 과정에서 문서 검색과 답변 생성 구조를 이해하고 구현 방향을 함께 논의했습니다.

기능 구조

검색 구조

키워드 기반 BM25와 의미 기반 FAISS 검색을 결합하고, LLM 리랭킹으로 관련 문서를 선별하는 구조를 사용했습니다.

서비스 구조

FastAPI 기반 웹 인터페이스와 LangGraph 워크플로우를 활용해 질문 재작성, 하위 질문 생성, 문서 검색, 답변 생성을 연결했습니다.

성과

2025 병무청·방위사업청·질병관리청 합동 데이터 분석 및 아이디어 공모전에서 대상, 1위를 수상했습니다.
