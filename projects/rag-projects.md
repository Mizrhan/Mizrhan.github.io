## Projects

### Army RAG

병무청 모집병 정보를 제공하는 RAG 챗봇 프로젝트입니다.  
LangGraph 기반 workflow, 하이브리드 검색, 리랭킹, FastAPI 웹 인터페이스, Docker Compose 기반 실행 구조를 포함합니다.

- Repository: <https://github.com/Mizrhan/army_rag>
- Note: Fork/협업 프로젝트라면 본인의 담당 역할을 반드시 명확히 적어야 합니다.

### Safety RAG

PDF 문서 기반 질의응답 RAG 시스템입니다.  
PDF 처리, Semantic Chunking, FAISS 벡터 저장소, Google Gemini/LangChain 기반 답변 생성을 포함합니다.

- Repository: <https://github.com/Mizrhan/safety_rag>
- Note: 공개 저장소에 인증키나 민감한 파일이 포함되어 있지 않은지 반드시 확인하세요.

## Portfolio Positioning

RAG 프로젝트는 다음 역량을 보여주는 데 적합합니다.

- Python 응용 개발
- 문서 데이터 처리
- 임베딩/검색/리랭킹 구조 이해
- FastAPI, Docker 기반 실행 환경 구성

다만 데이터 분석가 지원에서는 SQL/KPI/고객분석/모델링 프로젝트보다 뒤에 배치하세요.
