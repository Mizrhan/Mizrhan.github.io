시정 예측 머신러닝·딥러닝 연구

대한민국 주요 도시의 시정 데이터를 예측하기 위해 기상 데이터와 대기질 데이터를 통합하고, 불균형 데이터와 분포 변화 문제를 모델링과 원인 분석 관점에서 다룬 프로젝트입니다.

ASOSAirKoreaDataON SMOTECTGANLightGBM XGBoostSHAPWasserstein Distance

문제 정의

낮은 시정은 교통, 항공, 해상 안전과 직접 연결되지만 데이터에서 자주 나타나는 범주는 아닙니다. 그래서 전체 정확도만 높이는 방식보다, 중요한 희소 구간을 얼마나 잘 잡는지가 더 중요했습니다. 프로젝트는 기상 데이터와 대기질 데이터를 결합해 시정 예측 성능을 높이고, 성능이 달라지는 지역과 연도의 원인을 설명하는 데 초점을 맞췄습니다.

내 역할

데이터 파이프라인

  • Haversine 공식을 활용해 기상 관측소와 대기질 측정소를 지리적으로 매핑
  • ASOS와 AirKorea 데이터를 시공간 기준으로 병합
  • 선형 보간과 인접 관측소 대체를 결합한 계층적 결측치 처리 로직 구현

모델링과 검증

  • SMOTE와 CTGAN을 순차적으로 적용하는 데이터 증강 전략 설계
  • XGBoost, LightGBM, ResNet-like, FT-Transformer, DeepGBM 모델 구축
  • Bayesian Optimization과 Soft-Voting 앙상블 구조로 최종 성능 검증

특정 지역 최대 개선

지역별로는 최대 42%의 성능 향상이 확인됐습니다.

성능 하락을 어떻게 해석했는가

모든 지역에서 성능이 오른 것은 아니었습니다. 한 지역에서는 CSI가 0.3820에서 0.3521로 약 8% 하락했습니다. 이 결과를 단순 실패로 처리하지 않고, Validation과 Test 성능의 괴리를 확인한 뒤 Shapley Value와 Wasserstein Distance로 연도별 시정 범주 데이터의 분포 변화를 분석했습니다. 이 경험은 모델 성능 수치보다 성능이 달라진 이유를 설명하는 능력이 중요하다는 점을 보여줍니다.
