시정 예측 머신러닝·딥러닝 연구

대한민국 주요 도시의 시정 데이터를 예측하기 위해 기상 데이터와 대기질 데이터를 통합하고, 불균형 데이터의 증강을 통해, 모델의 안정성을 높였습니다. 모델링 이후, 기존 모델과의 성능 비교와 일부 지역의 성능 하락 원인 분석까지 전과정을 다룬 프로젝트입니다.

ASOSAirKoreaDataON SMOTECTGANLightGBM XGBoostSHAPWasserstein Distance

문제 정의

시정불량은 교통, 항공, 해상 안전에 치명적이지만 흔하게 발생하는 이벤트는 아닙니다. 따라서 시정 범주 전체의 예측 정확도만 높이는 방식보다, 중요한 희소 구간을 얼마나 잘 잡는지가 더 중요합니다. 해당 프로젝트는 기상 데이터와 대기질 데이터를 결합해 시정 예측 성능을 높이고, 성능이 달라지는 지역의 원인을 설명하는 데 초점을 맞췄습니다.

역할

이 프로젝트에서는 데이터 병합 구조 설계, 불균형 데이터 증강 전략, 모델 구축, 그리고 일반화 성능이 떨어진 지역의 원인 분석을 담당했습니다.

데이터 파이프라인

  • Haversine 공식을 활용해 기상 관측소와 대기질 측정소를 지리적으로 매핑했습니다.
  • ASOS 기상 데이터와 AirKorea 대기질 데이터를 시공간 기준으로 병합했습니다.

모델링과 검증

  • SMOTE와 CTGAN의 데이터 증강 성능을 비교하여 증강 모델을 선별하는 전략을 설계했습니다.
  • XGBoost, LightGBM, ResNet-like, FT-Transformer, DeepGBM 모델을 구축했습니다.
Key Contribution

성능 하락 지역의 원인 규명

단순히 평균 성능이 개선됐다는 결과에서 끝내지 않고, Validation 성능 대비 Test 성능의 하락 폭이 유독 크게 나타난 지역을 따로 확인했습니다. 이후 SHAP Value와 Wasserstein Distance를 활용해 어떤 변수와 분포 변화가 일반화 성능 저하에 영향을 주었는지 분석했습니다.

  • SHAP Value를 활용해 지역별 예측에 중요한 변수를 선별했습니다.
  • Wasserstein Distance로 연도별 데이터 분포 변화를 정량적으로 비교했습니다.

성능 하락을 어떻게 해석했는가

각 모델의 SHAP value 상위 변수들의 Wasserstein Distance를 비교한 결과, 성능이 하락한 지역의 정상시정 범주의 변수 분포가 TEST 연도 데이터에서 크게 변화했음을 포착했습니다. 이 결과를 바탕으로 해당 연도, 해당 지역의 기상관측장비 재원을 문의한 결과, 동일한 지역과 연도에서 장비의 교체가 진행되었음을 확인할 수 있었습니다. 따라서, 특정 지역의 성능 하락이 모델 자체의 실패라기보다, 시정 범주 데이터의 분포 변화와 관련되어 있음을 규명했고, 추후 모델링 성능의 향상을 위해, 동일한 장비로 측정한 데이터의 축적이 필요하다는 시사점을 도출했습니다.
