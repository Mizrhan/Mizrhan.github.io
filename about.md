# About

통계 기반의 문제 정의와 Python, SQL, R 분석 역량을 바탕으로 데이터를 다룹니다.  
관심사는 운영 데이터 분석, KPI 설계, 고객/마케팅 분석, 예측 모델 검증, 분석 결과의 의사결정 연결입니다.

## Working Style

- 데이터가 이미 깨끗하다고 가정하지 않고, 결측·이상치·기준 변화를 먼저 확인합니다.
- 모델 성능 수치만 보는 대신, 어떤 조건에서 오차가 커지는지 해석합니다.
- 대시보드와 리포트는 “보여주기”보다 “다음 행동을 결정하게 하기”를 목표로 설계합니다.
